# Rock_Paper_Scissors_5m
That game is a RPS game but in that code we have some extra characters

🎮 Rock, Paper, Scissors, Lizard, Spock
A fun variation of the classic Rock, Paper, Scissors game, inspired by The Big Bang Theory.
How to Play
Choose one of the following options:
*	🧱 Rock
*	
•	📄 Paper
•	✂️ Scissors
•	🦎 Lizard
•	🖖 Spock
The computer (Dalin AI) will also make a choice. The winner is determined according to the following rules:
Winning Rules
•	Scissors cuts Paper ✂️ > 📄
•	Paper covers Rock 📄 > 🧱
•	Rock crushes Lizard 🧱 > 🦎
•	Lizard poisons Spock 🦎 > 🖖
•	Spock smashes Scissors 🖖 > ✂️
•	Scissors decapitates Lizard ✂️ > 🦎
•	Lizard eats Paper 🦎 > 📄
•	Paper disproves Spock 📄 > 🖖
•	Spock vaporizes Rock 🖖 > 🧱
•	Rock crushes Scissors 🧱 > ✂️
If both players choose the same option, the round ends in a draw.
Scoring
•	Each win earns 1 point.
•	Draws do not affect the score.
•	Use the Restart button to start a new game at any time.
Good luck and have fun playing against Dalin AI! 🤖
